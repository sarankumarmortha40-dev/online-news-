# Online News Website

An Online News Website designed to make news more accessible and easy to understand.

This platform fetches news articles, translates them into regional languages, summarizes the content, and explains articles in simple terms—helping users quickly understand important information without language barriers.

---

## 🚀 Features

- 🌐 Translates news articles into regional languages  
- ✍️ Summarizes long news articles  
- 📖 Explains complex news in simple and easy-to-understand terms  
- 🗂 Category-wise news organization (World, Politics, Sports, Tech, Health, Entertainment, etc.)
- 🔁 Automated news updates using backend scripts
- 📊 Stores processed news data in JSON format

---

## 🧠 How the Project Works

1. News articles are fetched from a news API using backend scripts
2. Articles are processed for:
   - Translation
   - Summarization
   - Simple explanation
3. Processed data is stored in JSON files
4. Frontend pages display category-wise news content to users
